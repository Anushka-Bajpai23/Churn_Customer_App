#ChurnGuard
ChurnGuard is a web-based application built with Flask that provides real-time customer churn prediction. It uses a pre-trained machine learning model to analyze customer data and predict their likelihood of churning. 

Overview:
* The dashboard offers an intuitive interface where users can input customer details and receive an instant churn probability score, along with actionable insights and data visualizations to help with customer retention.
* The application is designed to be a simple, yet powerful tool for businesses to identify at-risk customers and take proactive measures to improve customer loyalty.

Technologies Used
-Backend: Python, Flask

-Machine Learning: Scikit-learn, Pandas, Joblib

-Data Visualization: Plotly

-Frontend: HTML, CSS (for styling), JavaScript (for dynamic chart rendering)
